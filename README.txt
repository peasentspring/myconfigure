1.copy .vimrc to your user directory.
2.exec 'vim' command
3.install vundle by command "git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim"
3.In command mode, type "PluginInstall".

Install 'YouCompleteMe' plug, you still need execute the following command:
cd ~/.vim/bundle/YouCompleteMe
./install.py --clang-completer
or check the git repo to see how to install ycm
